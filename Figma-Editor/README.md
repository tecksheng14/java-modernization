# Figma Integration Lab - With Editor Access

**Box Folder Link:** [https://ibm.ent.box.com/folder/373365438389](https://ibm.ent.box.com/folder/373365438389)

## 📋 Overview

This lab teaches you how to integrate IBM Bob with Figma when you have **editor access** to Figma files. You'll learn to use IBM Bob to interact with Figma designs, make modifications, and understand the Model Context Protocol (MCP) for Figma integration.

## 🎯 What You'll Learn

- Connect IBM Bob to Figma with editor permissions
- Use IBM Bob to read and modify Figma designs
- Understand Figma's API and MCP integration
- Automate design tasks with AI assistance

## ⏱️ Duration

**30-45 minutes**

## 📚 Prerequisites

### Required
- IBM Bob installed in your IDE
- Figma account (free or paid)
- Figma editor access to a design file

### Recommended Knowledge
- Basic understanding of Figma
- Familiarity with IBM Bob

## 🆘 Troubleshooting

**Problem: Can't access Figma file**
- Verify you have editor permissions
- Check your Figma account is logged in
- Ensure the file link is correct

**Problem: IBM Bob not connecting to Figma**
- Verify Figma API token is configured
- Check network connectivity
- Restart IBM Bob

**Problem: MCP errors**
- Review the MCP configuration in the guide
- Verify all prerequisites are installed
- Check the error message details
